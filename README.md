佳兴汽车综合管理系统
=====================

佳兴汽车综合管理系统是一款销售、库存、会员、股东分红、微信分销等综合管理系统，目前只完成了库存模块和基础设置功能

## 功能特性

- 库存管理(完成)
- 会员管理
- 收银开单
- 财务管理
- 员工绩效
- 股东分红
- 未完待续

## 运行环境

此项目是在 LNMP 环境下开发的，理论上也支持 LAMP 没有实际测试过

- 操作系统 - CentOS 7.4
- nginx
- Mysql 5.6
- PHP 7.0
